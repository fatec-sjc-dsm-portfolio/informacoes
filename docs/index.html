Informações e orientações sobre a criação do portfólio digital.

Requisitos de Informação

Os requisitos de informação delimitam o escopo do conteúdo que deve ser mostrado no portfólio.

Por se tratar de um portfólio profissional, o(a) aluno(a) deve acrescentar informações relevantes à sua trajetória acadêmica. A relevância deve considerar tanto o ponto de vista do aluno, como um profissional formação, como o da Faculdade de Tecnologia de São José dos Campos, como instituição formadora de profissionais qualificados para atuar no mercado de trabalho.

  • O portfólio deverá ser composto por, no máximo, três seções.
  • As seções deverão incluir informações tais como:
    • Apresentação pessoal.
    • Projetos acadêmicos.
    • Projetos profissionais.
    • Projetos pessoais.
    • Áreas de atuação.
    • Outras informações relevantes.
  • As tecnologias que porventura sejam citadas no escopo dos trabalhos que foram desenvolvidos, podem ser agrupadas em uma única seção (exemplo: ao final dos projetos, sintetizando a informação) ou em uma seção específica onde o(a) aluno(a) cita suas habilidades e conhecimentos tecnológicos.

ATENÇÃO: Na seção de apresentação pessoal, o(a) aluno(a) deve acrescentar uma foto compatível com o propósito do portfólio.

Restrições de Tecnologia

  • O portfólio deve ser confeccionado com tecnologias que permitam o processamento totalmente do lado do cliente (ex. HTML5, CSS3, JavaScript, arquivos estáticos de imagem png, etc.)
  • Caso o repositório do portfólio dependa de bibliotecas estáticas para renderização da interface, essas devem ser devidamente "instaladas" no portfólio (exemplo: pasta lib/) de forma que a interface não se torne desconfigurada no caso de quebra da referência da biblioteca (exemplo: mudança de link externo que provê o arquivo estático da biblioteca. )

Restrições de Dados

  • A totalidade dos dados necessários para a exibição do portfólio deve ser disponibilizada de modo estático no próprio repositório do portfólio.
